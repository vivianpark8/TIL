# 7. 목록
- 순서를 표시
- 순서 있는 목록과 순서 없는 목록이 있음

## 순서 있는 목록
1. 첫 번째
2. 두 번째
3. 세 번째

## 순서 없는 목록

- 첫 번째
- 두 번째
- 세 번째

- 하위 항목을 구성할 때 필요함

    - 1-2
